# Sesión 16- Separar cadenas de texto en Python

* La función **slice** de python nos permite separar los strings en substrings generando nuevas secuencias.

```python
my_string = 'platzi'

my_string[1:3] # la

my_string[1:] # latzi

my_string[1:6:2] # lti

my_string[::-1] # iztalp

#my_string[start:end:steps]
```
